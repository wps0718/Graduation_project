## 三、🔴 高优先级：MyBatis-Plus 自动填充未启用

### 问题描述

项目中大量 Entity 使用了 `@TableField(fill = FieldFill.INSERT)` 和 `@TableField(fill = FieldFill.INSERT_UPDATE)` 注解，但**未实现 MetaObjectHandler**。

**涉及的 Entity**（共17个）：
- User、Product、TradeOrder、ChatMessage、ChatSession
- Favorite、Review、Report、Notification
- Campus、College、Category、MeetingPoint、CampusAuth
- Banner、Notice、SearchKeyword、Employee

**影响**：
- `create_time`、`update_time` 字段无法自动填充
- 可能导致数据插入/更新失败或时间戳为 null

**修复方案**：

创建 `MyBatisMetaObjectHandler.java`：

```java
// 文件：src/main/java/com/qingyuan/secondhand/config/MyBatisMetaObjectHandler.java

package com.qingyuan.secondhand.config;

import com.baomidou.mybatisplus.core.handlers.MetaObjectHandler;
import org.apache.ibatis.reflection.MetaObject;
import org.springframework.stereotype.Component;

import java.time.LocalDateTime;

@Component
public class MyBatisMetaObjectHandler implements MetaObjectHandler {

    @Override
    public void insertFill(MetaObject metaObject) {
        this.strictInsertFill(metaObject, "createTime", LocalDateTime.class, LocalDateTime.now());
        this.strictInsertFill(metaObject, "updateTime", LocalDateTime.class, LocalDateTime.now());
    }

    @Override
    public void updateFill(MetaObject metaObject) {
        this.strictUpdateFill(metaObject, "updateTime", LocalDateTime.class, LocalDateTime.now());
    }
}
```

**注意事项**：
- 需要确保实体类的字段类型为 `LocalDateTime`（已符合）
- 如果使用 `Date` 类型，需要相应修改

### 方案B：小程序端改为URL参数（备选）

如果不想修改后端，可以修改小程序端的请求方式。

#### 步骤1：修改request.js支持URL参数

```javascript
// 文件：miniapp/utils/request.js

// 在 request 函数中添加对 GET 请求的 URL 参数处理
function request(url, method = 'GET', data = {}, options = {}) {
  const mergedOptions = {
    showLoading: true,
    ...options
  }
  
  // 对于 GET 请求，将 data 转换为 URL 参数
  if (method === 'GET' && data && Object.keys(data).length > 0) {
    const params = new URLSearchParams(data).toString()
    url = url + '?' + params
    data = {}
  }
  
  if (USE_MOCK) {
    return mockRequest(url, method, data, mergedOptions)
  }
  return realRequest(url, method, data, mergedOptions)
}
```

#### 步骤2：修改小程序调用

```javascript
// miniapp/pages/product/detail/detail.vue

// 修改前
await post('/mini/product/off-shelf', { productId: id })

// 修改后 - 改为 query 参数
await post('/mini/product/off-shelf?productId=' + id, {})
```

**注意**：这种方案需要逐个修改每个调用点，工作量较大，建议使用方案A。
