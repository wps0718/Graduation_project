## 🛠️ 环境准备（工欲善其事，必先利其器）

### 1. 后端服务启动
```bash
# 1. 确保MySQL服务运行（端口3306）
# 2. 确保Redis服务运行（端口6379）
# 3. 进入后端项目目录
cd G:\Code\Graduation_project

# 4. 启动后端服务
mvn spring-boot:run

# 或使用IDE直接运行 SecondhandApplication.java
# 后端默认端口：8080
```

### 2. 前端配置修改
```javascript
// miniapp/utils/request.js 第5行
const USE_MOCK = false  // 关闭Mock，使用真实接口
```

### 3. 微信开发者工具配置
1. 打开微信开发者工具
2. 进入 → 项目详情 → 本地设置
3. 勾选 ✅ **不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书**
4. 刷新项目

### 4. 调试工具准备

| 工具 | 用途 | 说明 |
|------|------|------|
| 微信开发者工具 | 小程序运行与调试 | 查看网络请求、控制台日志 |
| Postman/Apifox | 接口独立测试 | 验证后端接口可用性 |
| Navicat | 数据库查看 | 验证数据落库情况 |
| Chrome DevTools | Web端管理端调试 | 管理后台前端调试 |

### 5. 关键地址确认

| 服务 | 地址 | 说明 |
|------|------|------|
| 后端API | http://localhost:8080 | RESTful接口 |
| WebSocket | ws://localhost:8080/ws/chat | IM即时通讯 |
| 文件上传 | http://localhost:8080/uploads/ | 静态资源 |
| 小程序 | 微信开发者工具内置 | 模拟器/真机 |

## 🐛 常见问题排查

| 问题现象 | 可能原因 | 解决方案 |
|----------|----------|----------|
| 请求404 | 接口路径错误/后端未启动 | 检查后端服务+接口路径 |
| 请求401 | Token失效/未携带 | 重新登录获取Token |
| 请求500 | 后端异常 | 查看后端控制台日志 |
| 图片不显示 | 文件路径错误/未配置静态资源 | 检查upload配置 |
| WebSocket连接失败 | 端口不通/Token错误 | 检查WS地址+Token |
| 数据库操作失败 | 数据库未创建/连接失败 | 检查MySQL配置 |
